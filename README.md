# ToDo List Application

![screenshot](https://i.ibb.co/2hYJRbP/screely-1637592652877.png)

## Introduction
This is a simple ToDoList Application built using react.js

It can be used to add and delete todos from the list and it also stores previous session values.

### Setup
- Npm needs to be installed.
- run `npm start` for running the application
